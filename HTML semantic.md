# HTML 시맨틱 태그

시맨틱 태그는 단순히 구역을 나누는 `<div>`와 달리
의미를 담고 있어 더 쉽게 페이지 구조를 이해할 수 있게 해줍니다.

시맨틱 태그는 `<header>,<footer>,<main>,<section>`으로 구성됩니다.

먼저 `<header>`는 페이지의 머리말 부분을 나타냅니다.
보통 제목이나 로고 등이 들어갑니다.

다음으로 `<footer>`는 페이지의 바닥글 부분을 나타냅니다.
주로 연락처가 들어갑니다.

`<main>`은 문서의 주요 콘텐츠 영역을 나타냅니다.
페이지의 가장 핵심적인 내용이 들어가는 곳입니다.

마지막으로 `<section>`은 관련된 문서의 영역을 구분할 때 사용합니다.
주로 제목과 함께 쓰이는 경우가 많습니다.

이렇게 HTML 시맨틱 태그가 구성됩니다.
