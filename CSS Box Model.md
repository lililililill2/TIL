# CSS Box Model

Box Model은 요소가 화면에 표시될 때 차지하는 공간을 설명하는 기본 개념입니다.

요소는 크게 내용, 패딩, 보더, 마진으로 구성됩니다.

먼저 패딩은 내용과 보더 사이의 여백입니다
각 방향마다 다르게 여백을 지정할 수 있습니다

```css
div {
  padding: 10px 20px 30px 40px; /*안쪽 여백이고 여백 순서는 상,우,하,좌 순서입니다*/
}
```

다음으로 보더는 요소를 둘러싸는 테두리입니다.
두께나 색상, 스타일을 지정할수 있습니다.

```css
div {
  border: 5px solid red;
  /*5px 두께인 실선 모양의 빨간색 테두리*/
}
```

마지막으로 마진은 요소와 다른 요소사이의 바깥 여백입니다.
마진도 패딩과 같이 각 방향마다 다르게 여백을 지정할 수 있습니다.

```css
div {
  margin: 10px 30px 40px 60px; /*바깥쪽 여백이고 여백 순서는 똑같이 상,우,하,좌 순서입니다*/
}
```
